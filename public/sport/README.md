# “E路向前 云动2026”线上长跑活动方案
值此长征胜利90周年、电子科学学院建院65周年之际，让我们以奔跑的姿态，用坚实的脚步去丈量，在数字的云端呈现，致敬伟大的长征壮举、礼赞“电子铁军”精神！以奋斗的激情和昂扬的斗志，续写电子人的崭新篇章！

现面向电子科学学院学员二大队开展“E路向前 云动2026”线上长跑活动，制定活动方案如下：

## 一、活动主题
“E路向前 云动2026”线上长跑

## 二、活动时间及参加人员
- **活动时间**：2026年4月1日—4月30日
- **参加人员**：二大队全体学员、教职工及家属，诚邀志同道合的朋友们一同参与

## 三、活动地点
参与者可自由选择跑步路线和场所，如健身房、学校操场、校园道路、学校附近的公园等，确保安全即可。

## 四、活动内容
1. **数字征程奖**（个人赛道）：开展“数字长征25000里”数据接力，全大队总里程目标12500公里，活动结束按个人总里程排名。
2. **最强挑战奖**（个人赛道）：30天内累计里程达到200公里，按个人总里程排名。
3. **师生协作奖**（团体赛道）：以教研室为单位（导师+学生，每组3人），接力完成65公里次数最多者排名（次数相同按总里程）。
4. **全勤打卡奖**（个人赛道）：每次跑2KM或20分钟以上视为打卡成功，按总打卡数排名（打卡数相同按里程）。
5. **最具创意奖**（个人赛道）：跑出党徽、长征路线图、E、90、65、北斗等创意轨迹（仅限校外），大队评选。
6. **最佳叙事奖**（团体赛道）：2—3人小组（情侣/夫妻/亲子/家庭/好友），每周至少打卡1次（2KM或20分钟以上）；完赛提交1张照片+100字内“微故事”，大队评选。

## 五、相关要求
1. 注意保密纪律，校内使用运动软件仅记录里程，勿记录轨迹；校外注意安全，尽量结伴而行。
2. 每人只能选择一个赛道报名。
3. 量力而行，科学运动，身体不适立即停止，必要时就医，坚持“安全第一、健康至上”。
4. 结果以当日里程截图为准，截图需含用户名、日期、距离、时长；弄虚作假取消资格。

## 六、奖品设置
### 奖项设置
各奖项均设一、二、三等奖各1名；师生协作奖、最佳叙事奖团队成员均有奖；各赛道前十名获纪念奖。

### 奖品明细
- 数字征程奖一等奖：华为蓝牙耳机
- 最强挑战奖一等奖：小米手表
- 师生协作奖一等奖：外交官拉杆箱
- 全勤打卡奖一等奖：尤尼克斯天斧羽毛球拍
- 最具创意奖一等奖：雷蛇V3键盘
- 最佳叙事奖一等奖：富士胶卷相机（含3卷胶卷）
- 二等奖：海尔腰部按摩仪
- 三等奖：小米充电宝
- 纪念奖：富光运动水杯
